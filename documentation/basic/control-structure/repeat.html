A for loop for elements of the DOM. This for loop iterates over the items given to it Arrays with the items attribute. The DOM element receives the item via the item attribute. Additionally the dom element receives the index of the iteration, the index is via the property index of the dom element available. Ein For Loop für Elemente des DOM. Dieser For Loop iteriert über die ihm gegebenen items eines Arrays mit dem items Attribut. Das DOM-Element erhält das item über das item Attribut. Zusätzlich erhält das Dom-Element den Index der iteration, der Index ist über die Property index des Dom-Elements verfügbar. Element Element Attribute Attribut Description Beschreibung The array to be looped with the given variable and source Das Array das durchlaufen wird mit der Variable und der Quelle The index of the given item for that array Der Index von dem gegebenen Element des Array The length of the given array Die Länge der gegebenen Array A callback to be called when the list has been rendered Ein Callback der Aufgerufen wird, wenn die Liste gerendert wurde
