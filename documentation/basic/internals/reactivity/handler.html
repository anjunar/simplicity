Reactivity - Handler Reaktivität - Handler The active membrane makes it possible for properties to be tracked by an event handler. The "addEventHandler" function makes this possible, while the property, the element when Exiting the dom de-registers the handler and passes the callback. In addition, there is the method "fire" on the active membrane that triggers the handlers. Die aktive Membrane macht es möglich das Properties durch einen Event-handler getrackt werden können. Die Funktion "addEventHandler" macht dies möglich, dabei wird die Property, das Element, dass beim Verlassen des Doms den Handler de-registriert und der Callback übergeben. Zusätzlich gibt es die Methode "fire" an der aktiven Membrane das die Handler auslöst.
