Custom elements give developers the ability to extend HTML and create their own tags. Because custom elements are standards based they benefit from the Web's built-in component model. The result is more modular code that can be reused in many different contexts. Benutzerdefinierte Elemente geben Entwicklern die Möglichkeit, HTML zu erweitern und eigene Tags zu erstellen. Weil die Elemente auf Standards basieren und vom eingebauten Komponentenmodell des Webs. Das Ergebnis ist mehr modularer Code, der in vielen verschiedenen Kontexten wiederverwendet werden kann. With Custom Elements, web developers can create new HTML tags, beef-up existing HTML tags, or extend the components other developers have authored. The API is the foundation of web components. It brings a web standards-based way to create reusable components using nothing more than vanilla JS/HTML/CSS. The result is less code, modular code, and more reuse in our apps. Mit benutzerdefinierten Elementen können Webentwickler neue HTML-Tags erstellen, vorhandene HTML-Tags aufpeppen oder erweitern die Komponenten, die andere Entwickler verfasst haben. Die API ist die Grundlage von Webkomponenten. es bietet eine auf Webstandards basierende Möglichkeit, wiederverwendbare Komponenten mit nichts anderem als Vanilla JS/HTML/CSS zu erstellen. das Ergebnis ist weniger Code, modularer Code und mehr Wiederverwendung in unseren Apps.
