A native implementation of the select element. Via bind: model you can access the value in two-way mode be taken from the option element. Native select works well when there are only a few values about the options element there. If the values exceed a certain number of possibilities, then it is advisable to use the lazy select. Eine native implementierung des select Elements. Über bind:model kann im two-way modus auf den Wert vom option Element genommen werden. Natives select lässt sich gut anwenden, wenn es nur wenige Werte über das options Element gibt. Überschreiten die Werte eine bestimmte Anzahl von Möglichkeiten, dann ist es ratsam das lazy select zu benutzen.
