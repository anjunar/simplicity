Simplicity is a progressive Web framework to build interfaces. Attention was paid to the lines of code Keeping it as low as possible, this promises readability and performance. On additional libraries was added deliberately avoided in order to keep the level of abstraction low. Simplicity ist ein Progressive Web Framework um Oberflächen zu bauen. Dabei wurde darauf geachtet den umfang des Quellcodes möglichst gering zu halten, das verspricht Lesbarkeit und Leistung. Auf zusätzliche Bibliotheken wurde bewusst verzichtet, um das Abstraktionsniveau niedrig zu halten. The framework is divided into three parts, simplicity contains the basic functionality, simplicity mainly contains the components in Material Design and simplicity-parser is the standalone javascript parser. Das Framework ist in drei Teile aufgeteilt, simplicity enthält die Grund-Funktionalität, simplicity enthält hauptsächlich die Komponenten im Material Design und simplicity-parser ist der eigenständige Javascript Parser. As the name suggests, this is about simplicity. That's why there is no shadow-dom, no other libraries and only vanilla javascript. Also NPM with NodeJS is not needed, since ES6 modules are used. Wie der Name schon sagt, handelt es sich hier um die Einfachheit. Deswegen gibt es keinen Shadow-Dom, keine weiteren Bibliotheken und nur vanilla Javascript. Auch NPM mit NodeJS wird nicht benötigt, da ES6 Modules genutzt werden. The "bootstrap" function is there to start the application. It should be noted that there are two functions "bootstrap" exists. Once in the "simplicity" directory and once in the directory "simplicity". The bootstrap function in the "simplicity" directory simultaneously includes the "bootstrap" function from the "simplicity" directory. The options object can do the following parameter contain. Die Funktion "bootstrap" ist dafür da, die Anwendung zu starten. Dabei ist zu beachten, dass es zweimal die Funktion "bootstrap" gibt. Einmal im Verzeichnis "simplicity" und einmal im Verzeichnis "simplicity". Die Bootstrap funktion im Verzeichnis "simplicity" beinhaltet gleichzeitig die "bootstrap" Funktion aus dem Verzeichnis "simplicity". Das Options Object kann folgende Parameter enthalten.
  • app - The Path to the App
  • mode - The mode of the App (development or production)
  • library - The path to the Simplicity Library
  • shadowDom - Shadow Dom activation (Experimental)
  • preFetch - Loads all available Views (Experimental)
  • history - HTML5 Navigation or with Hashtag
  • app - Der Pfad zur Anwendung
