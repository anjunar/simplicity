Custom View with query parameters Eigene View mit Query Parametern This is a custom query parameters page. The path in the URL goes through the routes, stored in the root of the app directly to a javascript file. Dies ist eine benutzerdefinierte Seite mit Abfrageparametern. Der Pfad in der URL führt über die Routen, die in der Root der App gespeichert sind, direkt zu einer Javascript Datei. The resulting Query parameters can be found at the component with the property "queryParams" In this case it is the activeRoute that corresponds to the component. Die resultierenden Abfrageparameter finden Sie bei der Komponente mit der Eigenschaft "queryParams". In diesem Fall ist es die activeRoute, die der Komponente entspricht. The rest of the HTML file is not displayed Der rest der HTML Datei wird nicht angezeigt
