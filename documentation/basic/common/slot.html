Introduction Einleitung This extension of the slot element can also use the tag, selector or name as a selector. If none of these selectors is specified, a wildcard selector is selected automatically. Diese Erweiterung des Slot-Elements kann zusätzlich den Tag, Selektor oder Name als Selektor verwenden. Wird keines dieser Selektoren angegeben, wird automatisch ein Wildcard Selektor gewählt. The collection can be accessed with the index attribute if several elements exist Mit dem index Attribut kann auf die Collection zugegriffen werden, falls mehrere Elemente existieren The attribute implicit together with let can be used to declare a variable with which the let attribute can be accessed. Das Attribut implicit zusammen mit let lässt eine Variable deklarieren, mit der über das let Attribut zugegriffen werden kann. Usually source refers to the ContentChildren of the component, this can be changed with these Attribute. So ContentChildren of another component can be selected. Normalerweise verweist source auf die ContentChildren der Komponente, dies lässt sich ändern mit diesen Attribut. So können ContentChildren einer anderen Komponente gewählt werden.
