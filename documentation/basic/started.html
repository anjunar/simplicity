Getting started Erste Schritte Simplicity is a progressive Web framework to build interfaces. Attention was paid to the lines of code Keeping it as low as possible, this promises readability and performance. On additional libraries was added deliberately avoided in order to keep the level of abstraction low. Simplicity ist ein Progressive Web Framework um Oberflächen zu bauen. Dabei wurde darauf geachtet den umfang des Quellcodes möglichst gering zu halten, das verspricht Lesbarkeit und Leistung. Auf zusätzliche Bibliotheken wurde bewusst verzichtet, um das Abstraktionsniveau niedrig zu halten. Build management was avoided because the ES6 modules are used. Each view is lazy loaded and the other attached components as well. Thus the framework achieves a excellent page load latency. Auf Bildmanagement wurde verzichtet, da die ES6 Modules verwendet werden. Jede View wird lazy geladen und die weiteren angehängten Komponenten ebenfalls. Somit erreicht das Framework eine ausgezeichnete Latenz beim Laden von Seiten. The architecture has been kept close to the web component and HTML specification. This framework can therefore be seen as an extension of the Web Component specification. Die Architektur ist nahe an der Web-Component und HTML Spezifikation gehalten worden. Somit kann dieses Framework als eine Erweiterung der Web-Component Spezifikation gesehen werden. Reactivity is given by the choice of Components that are using onPush together with the Membrane that tracks changes. Die Reaktivität wird durch die Wahl der Komponenten gegeben, die onPush arbeiten, zusammen mit der Membran die Änderungen verfolgt. The code can be downloaded from the Github repository, the link is available in the toolbar above. Der Code kann vom Github Repository heruntergeladen werden, der Link ist oben in der Toolbar verfügbar.
