Meta Programming Meta Programmierung In computer science, metaprogramming is the creation of computer programs, the computer programs generate. The aim is to have the computer participate in the programming process in whole or in part to let. Metaprogrammierung ist in der Informatik die Erstellung von Computerprogrammen, die Computerprogramme erzeugen. Das Ziel dabei ist es, den Computer ganz oder teilweise an dem Prozess der Programmierung mitarbeiten zu lassen. Meta programming is done using a schema. Each model should have its own scheme own. In this case, the JSON schema standard is adhered to. The table or the form can then generate the columns or fields from the schema. Meta Programmierung wird mithilfe von einem Schema durchgeführt. Jedes Model sollte ein eigenes Schema besitzen. In diesen Fall wird sich an dem JSON Schema Standard gehalten. Die Tabelle oder auch das Formular können dann aus dem Schema die Spalten oder auch Felder generieren.
