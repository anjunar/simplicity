Meta Programming Meta Programmierung Compiler Kompiler The compiler is a central part of the framework and works together with the Dom parser. A separate HTML parser was not used because the performance of the Dom parser is too high. The compiler translates the dom into javascript to e.g. break down the context of the components. It is important to understand the compiler in order to provide plugins for the framework. Der Kompiler ist zentraler Bestandteil des Frameworks und arbeitet mit dem Dom-Parser zusammen. Es wurde extra auf einen eigenen HTML-Parser verzichtet da die Performance des Dom-Parser zu hoch ist. Der Kompiler übersetzt den Dom in Javascript um z.B. den Kontext der Komponenten aufzuschlüsseln. Es ist wichtig den Kompiler zu verstehen um Plugins für das Framework bereitzustellen.
