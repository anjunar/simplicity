This is a dialog that is rendered into the viewport. The dialog can be started either with the action attribute, or generated with the windowManager. The only important thing is the modal flag in the view that represents the dialog. Width and Height can also be given, otherwise the size of the dialog is calculated from the content. Dies ist ein Dialog, der in den viewport gerendert wird. Der Dialog kann entweder mit dem action Attribut, oder mit dem windowManager erzeugt werden. Wichtig ist nur das Flag modal in der View die den Dialog darstellt. Es kann ebenso Width und Height mitgegeben werden, ansonsten wird die Größe des Dialogs durch den Inhalt berechnet.
