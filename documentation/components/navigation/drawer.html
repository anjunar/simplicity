A mat drawer implementation to manage navigation. If the resolution falls below 800px, then the drawer is placed over the content. Otherwise the drawer is inserted from the left. Eine Mat Drawer implementation um Navigation zu bewerkstelligen. Wenn die Auflösung 800px unterschreitet, dann legt sich der Drawer über den Content. Ansonsten wird der Drawer eingeschoben von links.
