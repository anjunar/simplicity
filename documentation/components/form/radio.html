A native radio implementation with associated mat radio container to switch to material design. With the attribute bind: model, the attribute value can be linked to the selected property. Eine native Radio Implementierung mit zugehörigen mat-radio-container um auf Material Design zu wechseln. Mit dem Attribut bind:model lässt sich der Attributwert value auf den gewählten Property binden.
