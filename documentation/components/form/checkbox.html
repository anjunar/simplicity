Input elements of type checkbox are rendered by default as boxes that are checked (ticked) when activated, like you might see in an official government paper form. A checkbox allows you to select single values for submission in a form (or not). Eingabeelemente vom Typ Checkbox werden standardmäßig als Kästchen gerendert, die aktiviert (angekreuzt) sind, wenn aktiviert, wie Sie vielleicht in einem offiziellen Regierungsformular sehen. Ein Kontrollkästchen ermöglicht Ihnen die Auswahl Einzelwerte für die Einreichung in einem Formular (oder nicht). A DOMString representing the value of the checkbox. This is not displayed on the client-side, but on the server this is the value given to the data submitted with the checkbox's name. Ein DOMString, der den Wert des Kontrollkästchens darstellt. Dies wird nicht clientseitig angezeigt, sondern auf der Server Dies ist der Wert, der den mit dem Namen des Kontrollkästchens übermittelten Daten zugewiesen wird. With the bind:model Attribute you can bind the value of the Checkbox to a property of your choice. Mit dem bind: model Attribut können Sie den Wert der Checkbox an eine Eigenschaft Ihrer Wahl binden.
