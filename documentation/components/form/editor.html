A native Wysiwyg editor. Images, links, tables and a flexbox container can be integrated. On the server side it is recommended to search for scripts within the document and to deactivate them, since scripts can also be inserted without any problems. Ein nativer Wysiwyg Editor. Es lassen sich Bilder, Links, Tabellen und ein Flexbox Container einbinden. Auf der Server Seite wird empfohlen auf Scripte innerhalb des Dokumentes zu suchen und zu deaktivieren, da ohne Probleme auch Scripte eingefügt werden können.
