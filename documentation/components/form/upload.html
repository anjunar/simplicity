This is an image upload component to upload images. The object that is bound via bind: model contains the name, lastModified and data. Data is a base64 representation of the image. Dies ist eine Image Upload Komponente um Images hochzuladen. Das Object das über bind:model gebunden wird, beinhaltet den name, lastModified und data. Data ist eine base64 repräsentation des Images.
